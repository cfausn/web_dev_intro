Node.js is typically pretty good at accomplishing many things at once. It's event-based, so a lot can be done server-side then typical hosting. It also allows for MySQL integration which is excellent, and is generally pretty easy to use as it's a simple command-line tool. Because it's a command-line tool, Node.js can be hosted in a variety of different places (local tunnels, PAAS, Servers, Bare metal to name a few).

In comparison to other web-hosting technologies, there are a few disadvantages. The API for ndoe is also not stable, and tends to change relatively frequently. It is beneficial for those who which to scale easily however, and is also very easy to learn in comparison to other web development tools.
